# Stats401-Final

* This project is a visualization of Genshin Impact, specifically its Gacha system and abyss team composition.
* All graphs are built with D3.js. 
* Developed with React (Next.js)

### Setting up

* run `cors-anywhere`
  * needed if request data from miHoYo
* run `my-app`
  * `npm install`
  * `npm run dev`
